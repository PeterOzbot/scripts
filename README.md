# scripts
Admin and other helper scripts

# Node

package.json :
```
{
  "name": "hello-world",
  "version": "1.0.0",
  "description": "hello-world cli client",
  "bin": "index.js"
}
```

The executable script i.e index.js should starts with a line "#!/usr/bin/env node"

Pack/install :

npm pack

npm install -g hello-world-1.0.0.tgz
